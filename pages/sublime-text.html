Installation Guide for Sublime Text

01

Download the Sublime Text Installer

  • Open your browser (such as Microsoft Edge, Google Chrome, or Mozilla Firefox).
  • Go to the official Sublime Text website: https://www.sublimetext.com/
  • Click the Download for Windows button to get the installer file.
02

Run the Installer

  • Once the download is complete, open the folder where the file is saved (usually in the "Downloads" folder).
  • Double-click the Sublime Text Setup.exe file to start the installation.
03

Start the Installation Process

  • The installation window will appear. Click Next to continue.
  • Choose the installation location or leave it as default, then click Next.
  • Select additional options if needed, then click Next..
  • Click Install to begin the installation process.
04

Complete the Installation

  • Wait for the installation to finish.
  • Once completed, click Finish to close the installer.
