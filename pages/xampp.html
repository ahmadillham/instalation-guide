Installation Guide for XAMPP

01

Download the XAMPP Installer

  • Open your browser (such as Microsoft Edge, Google Chrome, or Mozilla Firefox).
  • Go to the official XAMPP website: https://www.apachefriends.org/
  • Click the Download for Windows button to get the installer file.
02

Run the Installer

  • Once the download is complete, open the folder where the file is saved (usually in the "Downloads" folder).
  • Double-click the XAMPP installer (.exe) file to start the installation.
  • If prompted by User Account Control (UAC), click Yes to allow installation.
03

Start the Installation Process

  • The XAMPP Setup Wizard will open. Click Next to continue.
  • Select the components you want to install. By default, all components are selected, but you can uncheck those you don't need. Click Next.
  • Choose the installation folder (default is C:\xampp). Click Next..
  • Click Next again to begin the installation.
04

Complete the Installation

  • Wait for the installation to finish. It may take a few minutes.
  • Once completed, click Finish to close the installer.
  • If prompted, check the option to start the XAMPP Control Panel and click Finish.
05

Start and Test XAMPP

  • Open the XAMPP Control Panel (if not opened automatically).
  • Start the Apache and MySQL services by clicking the Start buttons next to them.
  • Open your browser and type http://localhost/ in the address bar to check if XAMPP is running correctly.
