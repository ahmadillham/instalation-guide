Installation Guide for Microsoft Edge

01

Download the Edge Installer

  • Open an existing browser (such as Google Chrome or Mozilla Firefox).
  • Go to the official Microsoft Edge website: https://www.microsoft.com/edge
  • Click the Download button and select the Windows version.
02

Run the Installer

  • Once the download is complete, open the folder where the file is saved (usually in the "Downloads" folder).
  • Double-click the MicrosoftEdgeSetup.exe file to start the installation
03

Start the Installation Process

  • If a confirmation window appears, click Yes to allow the installation
  • The installation will begin automatically. Wait for it to finish.
04

Complete the Installation

  • Once the installation is finished, Microsoft Edge will open automatically.
  • If not, you can find it in the Start menu or on the desktop.
