Installation Guide for VS Code

01

Download the VS Code Installer

  • Open your browser (such as Microsoft Edge, Google Chrome, or Mozilla Firefox).
  • Go to the official Visual Studio Code website: https://code.visualstudio.com/
  • Click the Download for Windows button to get the installer.
02

Run the Installer

  • Once the download is complete, open the folder where the file is saved (usually in the "Downloads" folder).
  • Double-click the VSCodeSetup.exe file to start the installation
03

Start the Installation Process

  • The installation window will appear. Click Next to continue.
  • Read and accept the license agreement, then click Next.
  • Choose the installation location or leave it as default, then click Next.
  • Select additional tasks (such as adding VS Code to the system path or creating a desktop shortcut), then click Next.
  • Click Install to begin the installation.
04

Complete the Installation

  • Wait for the installation to finish.
  • Once completed, check the Launch Visual Studio Code option if you want to open it immediately.
  • Click Finish to close the installer.
