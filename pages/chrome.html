Installation Guide for Chrome

01

Download the Chrome Installer

  • Open an existing browser (such as Microsoft Edge or Mozilla Firefox).
  • Go to the official Google Chrome website: https://www.google.com/chrome/
  • Click the Download Chrome button to download the installation file.
02

Run the Installer

  • Once the download is complete, open the folder where the file is saved (usually in the "Downloads" folder).
  • Double-click on the installer file Firefox Setup.exe to start the installation process.
03

Start the Installation Process

  • The installation window will appear. Click Next or Install (depending on the version).
  • Select Standard Installation to use the default settings, or Custom Installation if you want to change the installation location.
  • Click Next until the installation process begins.
04

Complete the Installation

  • Wait for the installation to finish.
  • Once completed, check the Launch Firefox now option to open Mozilla Firefox immediately.
  • Click Finish to close the installation window.
